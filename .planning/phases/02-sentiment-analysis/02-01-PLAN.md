   - Return average sentiment (-1 to +1)

2. map_sentiment_to_risk(sentiment_score) -> float:
   - Input: sentiment_score (-1 to +1)
   - Output: risk contribution (0.0 to 0.8)
   - Formula: 0.4 * (1 - normalized) where normalized = (score + 1) / 2
   - Negative sentiment (-1) -> risk = 0.4
   - Positive sentiment (+1) -> risk = 0.0

3. calculate_combined_risk(volatility, sentiment_score) -> float:
   - Formula: 0.6 * volatility + 0.4 * (1 - normalized_sentiment)
